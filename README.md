# NHS Relational Database System

This repository contains the SQL files and MySQL Workbench output screenshots for the CPU4-103 Introduction to Database summative assessment.

## Project
NHS Relational Database System: Design, Implementation and Security

## SQL Files
- `01_create_database_and_tables.sql` creates the database tables, primary keys, foreign keys, and constraints.
- `02_insert_sample_data.sql` inserts sample NHS records.
- `03_advanced_queries.sql` contains JOIN, aggregate, SELECT, UPDATE, and reporting queries.
- `04_stored_procedure.sql` creates and runs the stored procedure `GetDoctorSummary`.
- `05_trigger_and_audit.sql` creates the appointment deletion trigger and audit table.
- `06_security_roles_and_hashing.sql` contains roles, privileges, hashing, and prepared statement examples.
- `07_run_all_scripts.sql` shows the correct running order.

## Folders
- `sql/` contains all SQL code files.
- `screenshots/` contains MySQL Workbench output evidence.
